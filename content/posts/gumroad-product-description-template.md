Your Gumroad description is doing 3 jobs at once:

1) convincing a buyer, 2) reducing refunds, and 3) helping search engines understand the page.

Here’s a copy-paste template that works well for **Notion templates, prompt packs, and cheat sheets**.

## Gumroad description template (copy/paste)

**One-line promise (keyword + outcome)**
> A *[Notion template / prompt pack]* to help *[audience]* get *[result]* in *[timeframe]*.

**Who it’s for**
- *[Audience 1]*
- *[Audience 2]*
- *[Audience 3]*

**What you get (be exact)**
- *[File/page 1]*
- *[File/page 2]*
- *[File/page 3]*

**How it works (3 steps)**
1) *[Step 1]*
2) *[Step 2]*
3) *[Step 3]*

**Setup time + requirements**
- Setup: *[2 minutes / 15 minutes]*
- Requires: *[Free Notion account / Notion + PDF reader]*

**FAQ (keep it short)**
- **Do I need Notion paid?** *[Answer]*
- **Can I duplicate instantly?** *[Answer]*
- **Is this beginner-friendly?** *[Answer]*

**One honest constraint (build trust)**
- This won’t *[magically do X]* — it will *[do Y]*.

**Related products (optional cross-links)**
If you have more products in the same problem space, link them here to increase internal traffic.
