If your Shorts results feel random, it’s usually because you’re not tracking the few numbers that actually explain *why* a video won.

This post gives you:
- the **minimum metrics** worth tracking
- a **simple review workflow** (weekly + monthly)
- a **copy‑paste template** you can use in Notion or a spreadsheet

## The only question analytics should answer
When a Short performs well, you want to know:

1) **Did people stop and watch?** (packaging + hook)
2) **Did they stay?** (structure + pacing)
3) **Did they act?** (CTA + value)

Everything you track should map to one of these.

## 1) Track these metrics (minimum viable)
You don’t need 20 columns. Start with these:

### A) Views
- **Views**: raw reach. Useful, but not diagnostic by itself.

### B) Retention proxy
- **Average view duration (AVD)** (or average watch time)
- **% viewed** (if you can see it)

Why: This is the fastest signal your hook + pacing works.

### C) Engagement intent
- **Likes** (light signal)
- **Comments** (strong signal)
- **Shares** (strongest signal)
- **Saves** (if applicable)

Why: Shares/saves often correlate with “this helped me” content.

### D) Click intent (optional)
- **Profile visits**
- **Link clicks** (if you have a link)

Only track this if you’re actively driving traffic.

## 2) Add these context fields (so you can learn)
Numbers without context don’t teach you anything. Add these columns:

- **Title / Hook** (the first line you said)
- **Topic** (1–2 words)
- **Format** (tutorial / list / story / reaction / comparison)
- **Length** (15s / 30s / 60s)
- **CTA type** (follow / comment / link / none)
- **Notes** (what you tried)

With just this, you can start spotting patterns like:
- “Stories do better at 45–60s than 15s.”
- “List formats get more saves, tutorials get more shares.”

## 3) A simple weekly review (10 minutes)
Once a week, answer these:

1) **Top 3 Shorts by views** — what do they have in common?
2) **Top 3 Shorts by retention** — what’s the hook + pacing pattern?
3) **Top 3 Shorts by shares/saves** — what value is being shared?
4) **1 thing to repeat next week**
5) **1 thing to stop doing**

Write the answer in 5 bullets. That’s it.

## 4) Monthly review (make one decision)
Once a month, pick *one* lever to focus on:
- better hooks
- tighter scripts
- clearer topic positioning
- more consistent format

If you pick multiple, you won’t improve any.

## 5) Copy‑paste tracking template
Use these columns:

- Date
- Platform
- Video URL
- Topic
- Format
- Length (s)
- Hook (first line)
- Views
- Avg view duration
- % viewed
- Likes
- Comments
- Shares
- Saves
- CTA type
- Notes

If you want to start even smaller: track **Views + Avg view duration + Shares**.
